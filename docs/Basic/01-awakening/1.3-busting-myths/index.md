# 1.3 破除迷信：不懂英文、数学不好也能写代码吗？

在开始动手之前，我想先帮你清除一些心理障碍。

这些年，我遇到过太多人，一听到"编程"两个字就摇头：

> "我数学不好，肯定学不会。"
> "我英语太差了，代码都是英文的。"
> "要懂那么多技术名词，太难了。"

如果你也有这些想法，别担心。

**这些都是迷信。**

在 AI 编程时代，这些"门槛"已经被大大降低，甚至消失了。


## 本章你将学到

| 小节 | 迷信 | 真相 |
|------|------|------|
| 1.3.1 | "编程需要数学很好" | 日常编程 95% 用不到高深数学 |
| 1.3.2 | "要先学好英文" | 中文 Prompt 完全可行 |
| 1.3.3 | "需要懂很多技术名词" | 边做边学，AI 会解释 |
| 1.3.4 | "Vibe Coding 只是玩具" | 有边界，但能做很多事 |
| 1.3.5 | "AI 会抢走程序员的工作" | 协作而非替代 |
| 1.3.6 | "AI 生成的代码总是完美的" | AI 会犯错，需要验证 |


## 为什么要先破除迷信？

因为**心理障碍比技术障碍更难跨越**。

很多人不是学不会，而是还没开始就把自己吓退了。

我见过数学只考 60 分的人，用 AI 做出了漂亮的网站。
我见过从没写过代码的 HR，用 AI 做了一个面试排班工具。
我见过英语四级没过的创业者，用 AI 做出了产品原型。

他们的共同点是：**先试了，再说。**

这一章，我要帮你卸下心理包袱。

看完这章，你会发现：原来这些"门槛"，根本不是门槛。

但如果你对自己有任何怀疑，请认真读完。**信心，是学习最好的燃料。**
