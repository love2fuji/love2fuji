# 写给不同的你

这份教程是写给"所有人"的。但"所有人"太抽象了，让我具体说说。

![五类读者画像](/images/5%20types.png)


## 写给我爸妈：不懂代码也能用 AI 做东西

如果你是我爸妈这样的人——

- 从没接触过编程
- 看到代码就头疼
- 觉得"这是年轻人的事"

我想告诉你：**时代变了。**

过去，编程确实需要学很多东西。但现在，AI 可以帮你处理那些复杂的部分。

你需要做的，只是**把你想要的东西说清楚**。

就像你告诉装修师傅"我想要一个亮堂的客厅"，师傅会帮你实现。你不需要自己砌墙、接电线。

这份教程会教你怎么和 AI 这个"师傅"沟通。


## 写给文科生/商科生：表达能力就是你的超能力

你可能一直觉得"编程是理科生的事"。

但 Vibe Coding 的核心能力是什么？

**是表达。是把想法说清楚。**

这恰恰是你的强项。

- 你擅长写作，能把复杂的事情说明白
- 你理解用户，知道人们真正想要什么
- 你懂沟通，能和 AI 进行有效的对话

在 AI 编程时代，**文科生的优势比你想象的大得多**。

这份教程会帮你把这些优势转化为实际的作品。


## 写给理工科学生：从写代码到审代码的角色转变

如果你有编程基础，你可能会问：

> "我都会写代码了，为什么还要学这个？"

因为 AI 正在改变程序员的工作方式。

过去，你是"代码的生产者"——一行一行地写。

现在，你正在变成"代码的指挥官"——告诉 AI 做什么，然后审查它的输出。

**核心技能正在迁移**：从"写得出来"到"说得清楚 + 判断对错"。

这份教程会帮你：

- 理解这种转变意味着什么
- 学会如何高效地与 AI 协作
- 为进阶版的全栈开发打下基础


## 写给职场人士：从提需求到自己验证的效率革命

你可能不是程序员，但你一定有过这样的经历：

- 想要一个小工具，但技术部门排期到三个月后
- 有个想法，但不知道技术上能不能实现
- 想快速验证一个方案，但没人帮你做
- 每周要从多个 Excel 里汇总数据，重复劳动让人崩溃
- 想做个数据可视化报告，但不会写代码

现在，你可以**自己动手**了。

不是让你变成程序员，而是让你：

- 能快速做出一个原型，验证想法
- 能用自然语言做数据分析，生成图表和报告
- 能自己写自动化脚本，告别重复劳动
- 能和技术同事更有效地沟通

这份教程会给你这个能力。


## 写给创业者：快速验证想法的新武器

创业最怕什么？

**花了三个月开发，结果没人用。**

Vibe Coding 能帮你在**几小时甚至几分钟内**做出一个能跑的原型。

- 快速测试市场反应
- 低成本试错
- 把精力放在真正重要的事情上

这份教程会教你用最小的投入，最快地验证你的想法。


## 不同背景的阅读建议

根据你的背景，这里是一些阅读重点建议：

### 如果你是文科生/商科生

你的语言表达能力是优势！建议重点关注：
- **第三章 技法**：学会写好 Prompt，这是你的强项
- **第四章 实战**：亲手做一个项目，建立信心
- 第一、二章可以快速浏览，理解核心概念即可

### 如果你是理工科学生

你可能已经有编程基础。建议：
- **第一章**：快速了解 Vibe Coding 理念，理解角色转变
- **第二、三章**：重点学习产品思维和 Prompt 工程方法
- 第四章可以快速完成，重点关注 Debug 技巧

### 如果你是职场人士

时间宝贵，建议优先：
- **第二章 心法**：学会 JTBD 和 MVP 思维，这对工作也有用
- **第四章 实战**：做一个解决工作痛点的工具
- 第五章的部署功能可以暂时跳过，等有需要再学

### 如果你是零基础学习者

不用担心，按顺序来就好：
- 从第一章开始，一步一步走
- 第四章可以选择更简单的项目
- 遇到不懂的，直接问 AI！

每个人的起点不同，但终点是一样的——**做出你想做的东西**。
