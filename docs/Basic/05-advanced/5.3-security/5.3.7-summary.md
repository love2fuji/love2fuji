# 5.3.7 本节要点总结

恭喜你完成了安全章节的学习。现在你已经掌握了 AI 编程时代最重要的安全知识。

## 三大禁令

记住这三条底线，可以避免 90% 的安全问题：

| 禁令 | 说明 |
|------|------|
| ❌ 不告诉 AI 密码/密钥 | 即使是"帮我检查格式对不对"也不行 |
| ❌ 不把密钥写在代码里 | 使用 .env 文件和环境变量 |
| ❌ 不上传 .env 文件 | 确保 .gitignore 配置正确 |

## 两个好习惯

1. **安装新库前先验证**：去 npmjs.com 或 pypi.org 确认包存在且可信
2. **提交前快速检查**：搜索 `password`、`secret`、`key` 等关键词

## 一个核心原则

::: tip 核心原则
**AI 生成的代码，最终责任在你。**

保持审查习惯，不盲目信任 AI 的输出。
:::
