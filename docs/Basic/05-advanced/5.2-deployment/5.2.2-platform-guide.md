# 5.2.2 部署平台选择指南

部署平台有很多，但对于初学者来说，选择标准很简单：**免费、简单、能用**。

::: warning 中国大陆用户请注意
Vercel 和 Netlify 的默认域名（`*.vercel.app` / `*.netlify.app`）在中国大陆被 DNS 污染，直接访问会失败或非常慢。

**建议**：大陆用户优先使用 Zeabur，它的免费域名（`*.zeabur.app`）可以正常访问。
:::

## 2025 年主流部署平台对比

| 平台 | 免费额度 | 大陆访问 | 难度 | 最适合 |
|------|---------|---------|------|-------|
| **Zeabur** | $5/月额度 | 正常 | 简单 | **大陆用户首选** |
| **Vercel** | 100GB带宽/月 | 需代理 | 简单 | 海外用户、Next.js 项目 |
| **Netlify** | 100GB带宽/月 | 需代理 | 简单 | 静态网站 |
| **帽子云** | 完全免费 | 正常 | 简单 | 大陆用户、静态网站 |
| **EdgeOne Pages** | 公测免费 | 正常 | 中等 | 腾讯云用户 |
| **GitHub Pages** | 完全免费 | 不稳定 | 中等 | 个人主页、文档站 |

## 2025 年免费额度详情

### Zeabur（推荐）

- 免费试用计划：每月 $5 额度
- 资源定价：内存约 $0.00025/GB-分钟（约 $10.8/GB-月）
- vCPU：免费
- 网络流量：$0.10/GB
- 特点：国人团队、中文客服、支持支付宝

对于待办清单这样的小项目，免费额度完全够用。

### Vercel

- Hobby 计划（免费）：
  - 100GB 快速数据传输/月
  - 1M Edge 请求/月
  - 4 小时 CPU 时间/月
  - 最多 200 个项目
- 限制：仅限非商业用途

### Netlify

- 免费计划：
  - 100GB 带宽/月
  - 300 分钟构建时间/月
  - 支持拖拽上传

## 如何选择

```
你在中国大陆吗？
├── 是 → 使用 Zeabur（首选）或帽子云
└── 否 → 使用 Vercel（首选）或 Netlify
```

如果你不确定，跟着本教程使用 **Zeabur** 即可。它是目前对大陆用户最友好的选择，操作也足够简单。

## 本教程的选择

| 场景 | 推荐平台 | 原因 |
|------|---------|------|
| 大陆用户 | Zeabur | 免费域名可正常访问，中文界面 |
| 海外用户 | Vercel | 最流行、功能强大、自动检测 |
| 纯静态网站 | 帽子云 / Netlify | 完全免费、操作简单 |
| 已有域名 | Vercel + Cloudflare | 性能最好 |

::: tip 不用纠结
对于你的第一个项目，选哪个平台差别不大。先跑起来最重要，以后可以随时换。
:::
