# 5.6.3 你现在能做什么

经过本节阅读，你将获得：
- 完成基础版后的能力清单
- 25+ 个按场景分类的项目建议
- 每个项目的核心挑战和参考方法
- 30 天持续练习计划

## 完成基础版后，你具备的能力

| 能力维度 | 具体表现 | 可以做什么 |
|---------|---------|-----------|
| **产品思维** | 能用 JTBD、MVP 思维规划项目 | 清晰定义"为谁做、做什么、为什么" |
| **提示词工程** | 能用 S.C.A.F.F. 框架与 AI 高效沟通 | 让 AI 准确理解需求，减少返工 |
| **前端开发** | 能做出包含增删改查的个人工具 | 静态网站、表单交互、数据展示 |
| **本地存储** | 能使用 localStorage 保存数据 | 数据刷新不丢失，换设备需重新输入 |
| **版本管理** | 能保护代码不丢失 | 大胆实验，改崩了随时回退 |
| **部署上线** | 能把作品发布到互联网 | 分享给朋友，获得真实反馈 |
| **安全意识** | 知道如何避免常见的安全坑 | 保护自己的账户和隐私 |
| **迭代能力** | 能根据反馈持续改进 | 让作品越来越好用 |

## 项目建议：按场景分类

以下项目都可以用基础版学到的方法完成。挑一个你感兴趣的开始吧。

### 场景一：个人效率工具

这类项目帮你管理时间、任务和个人事务。

| 项目 | 核心功能 | 学习价值 | 难度 |
|------|---------|---------|------|
| **番茄钟** | 25分钟倒计时 + 休息提醒 + 专注记录 | 学习定时器和时间处理 | ⭐⭐ |
| **习惯打卡器** | 每日打卡 + 连续天数 + 日历视图 | 学习日期处理和数据统计 | ⭐⭐ |
| **简易记账本** | 收支记录 + 分类统计 + 月度汇总 | 学习数字计算和数据展示 | ⭐⭐ |
| **读书笔记** | 书籍管理 + 笔记记录 + 标签分类 | 学习文本处理和搜索过滤 | ⭐⭐⭐ |
| **密码生成器** | 自定义长度 + 字符类型选择 + 一键复制 | 学习随机数和剪贴板操作 | ⭐ |
| **倒计时工具** | 目标日期设置 + 剩余天数显示 | 学习日期差值计算 | ⭐ |
| **每日一句** | 随机名言 + 收藏功能 + 分享按钮 | 学习数组随机和分享API | ⭐ |

**参考方法**：使用待办清单的开发流程，先用 S.C.A.F.F. 框架描述需求，再分层实现。

### 场景二：数据分析可视化

这类项目帮你整理和展示数据。虽然基础版不涉及复杂的数据库，但可以用本地数据做简单的分析展示。

| 项目 | 核心功能 | 学习价值 | 难度 |
|------|---------|---------|------|
| **个人支出分析** | 手动输入数据 + 饼图/柱状图展示 | 学习图表库使用（如 Chart.js） | ⭐⭐⭐ |
| **阅读统计** | 阅读时长记录 + 周/月汇总图表 | 学习时间统计和数据聚合 | ⭐⭐ |
| **体重追踪** | 每日记录 + 趋势折线图 | 学习折线图绑定数据 | ⭐⭐ |
| **学习时间统计** | 科目分类 + 时长记录 + 占比分析 | 学习多维度数据处理 | ⭐⭐⭐ |
| **问卷结果展示** | 静态数据输入 + 可视化报告页 | 学习数据可视化基础 | ⭐⭐ |

**参考方法**：让 AI 帮你集成 Chart.js 等图表库，你只需要提供数据结构和展示需求。

### 场景三：自动化脚本/批处理

这类项目帮你减少重复劳动。基础版主要做浏览器端的自动化，复杂的本地文件操作需要进阶版。

| 项目 | 核心功能 | 学习价值 | 难度 |
|------|---------|---------|------|
| **文本格式转换** | 文本输入 + 格式转换（大小写、去空格等） | 学习字符串处理 | ⭐ |
| **JSON 格式化** | JSON 输入 + 美化/压缩 + 语法检查 | 学习 JSON 解析和格式化 | ⭐⭐ |
| **Markdown 预览器** | 实时编辑 + 即时预览 | 学习 Markdown 解析库 | ⭐⭐ |
| **链接批量提取** | 粘贴文本 + 自动提取所有链接 | 学习正则表达式基础 | ⭐⭐ |
| **单位换算器** | 长度/重量/温度等单位转换 | 学习数学计算和单位换算 | ⭐ |

**参考方法**：这类工具通常输入输出明确，非常适合用 Vibe Coding 快速实现。

### 场景四：给家人/朋友做的工具

这类项目帮你解决身边人的实际问题，是获得真实反馈的最佳方式。

| 项目 | 核心功能 | 适合谁 | 难度 |
|------|---------|-------|------|
| **家庭菜单规划** | 每周菜单 + 食材清单生成 | 负责做饭的家人 | ⭐⭐ |
| **药物提醒** | 用药记录 + 定时提醒 | 需要定期服药的长辈 | ⭐⭐ |
| **生日提醒** | 亲友生日管理 + 倒计时提醒 | 容易忘记重要日子的人 | ⭐⭐ |
| **简易相册** | 图片上传展示 + 时间轴排列 | 想分享照片的家人 | ⭐⭐⭐ |
| **购物清单** | 共享清单 + 已购标记 | 一起生活的家人 | ⭐⭐ |
| **家庭记账本** | 共同支出记录 + 分类统计 | 合租室友或小家庭 | ⭐⭐⭐ |

**参考方法**：先和使用者聊聊他们的痛点（用灵魂三问），再用 MVP 思维确定核心功能。

### 场景五：个人展示/创意项目

这类项目展示你的个人品牌或创意想法。

| 项目 | 核心功能 | 学习价值 | 难度 |
|------|---------|---------|------|
| **个人主页** | 自我介绍 + 作品集展示 + 联系方式 | 学习页面布局和样式设计 | ⭐⭐ |
| **在线简历** | 响应式简历页面 + 下载PDF功能 | 学习响应式设计和打印样式 | ⭐⭐ |
| **作品集网站** | 项目展示 + 详情页 + 技术栈说明 | 整合所有学习成果 | ⭐⭐⭐ |
| **小游戏：猜数字** | 随机数生成 + 猜测反馈 + 记录次数 | 学习游戏逻辑设计 | ⭐ |
| **小游戏：记忆翻牌** | 卡片匹配 + 计时 + 最佳记录 | 学习状态管理和动画 | ⭐⭐⭐ |
| **表白网页** | 动画效果 + 音乐播放 + 个性化内容 | 学习CSS动画和音频API | ⭐⭐ |

**参考方法**：创意项目不需要复杂功能，重点是视觉效果和交互体验。让 AI 帮你实现炫酷的 CSS 动画。

## 如何选择你的下一个项目

不知道选哪个？用这个决策流程：

```
你现在最想解决什么问题？
│
├── 工作/学习效率低 → 番茄钟、习惯打卡器
├── 数据杂乱无章 → 记账本、阅读统计
├── 重复劳动太多 → 文本格式转换、链接提取
├── 想帮助家人朋友 → 药物提醒、生日提醒
└── 想展示自己 → 个人主页、在线简历
```

**核心原则**：选一个你真正会用的项目，而不是"看起来最酷"的项目。

## 30 天练习计划

如果你想持续提升，可以尝试这个计划：

| 周次 | 目标 | 建议项目 |
|------|------|---------|
| 第 1 周 | 复习巩固 | 给待办清单添加 2-3 个新功能 |
| 第 2 周 | 独立完成 | 选一个 ⭐ 难度的项目从零做起 |
| 第 3 周 | 挑战进阶 | 选一个 ⭐⭐ 难度的项目 |
| 第 4 周 | 分享展示 | 整理作品集，写 README，分享给朋友 |

**每个项目的标准流程**：
1. 用灵魂三问定义需求
2. 用 S.C.A.F.F. 框架写第一个 Prompt
3. 分层迭代实现功能
4. 部署上线，收集反馈
5. 整理成作品集

## 本节要点

- 基础版让你具备了做个人工具的完整能力
- 25+ 个项目建议覆盖效率、数据、自动化、家人、创意五大场景
- 选项目的关键是"你真正会用"，而不是"看起来最酷"
- 30 天计划帮你持续练习，巩固方法论

每做一个项目，你就更熟练一点。

从第一个待办清单开始，你已经证明了自己能做出东西。现在，去做更多。
