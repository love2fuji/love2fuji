# E.3 内容创作类（10 个）

这些项目适合喜欢写作、创作的人。用 AI 帮你搭建自己的创作平台。


## 1. 个人博客

**难度**：⭐⭐（基础）  
**预计时长**：2-3 小时  
**核心技术点**：页面布局、Markdown 渲染

**功能描述**：
- 文章列表页
- 文章详情页
- 分类/标签
- 简洁的阅读体验

**学到什么**：多页面结构、内容展示


## 2. 作品集展示

**难度**：⭐⭐（基础）  
**预计时长**：2-3 小时  
**核心技术点**：图片展示、布局设计

**功能描述**：
- 作品卡片展示
- 点击查看详情
- 分类筛选
- 响应式布局

**学到什么**：网格布局、图片优化


## 3. 写作计时器

**难度**：⭐（入门）  
**预计时长**：1 小时  
**核心技术点**：计时器、字数统计

**功能描述**：
- 设定写作时长
- 专注模式（全屏写作）
- 实时字数统计
- 时间到了提醒

**学到什么**：专注模式实现、计时逻辑


## 4. 每日写作提示

**难度**：⭐（入门）  
**预计时长**：30 分钟 - 1 小时  
**核心技术点**：随机内容、日期判断

**功能描述**：
- 每天显示一个写作提示/灵感
- 收藏喜欢的提示
- 可以自己添加提示

**学到什么**：日期处理、内容管理


## 5. 微博/推特风格发布器

**难度**：⭐⭐（基础）  
**预计时长**：1-2 小时  
**核心技术点**：字符限制、时间线展示

**功能描述**：
- 限制字数的输入框
- 发布后显示在时间线
- 显示发布时间
- 删除功能

**学到什么**：字符限制、时间格式化


## 6. 简易 Wiki

**难度**：⭐⭐⭐（进阶）  
**预计时长**：3-4 小时  
**核心技术点**：内容编辑、链接关联

**功能描述**：
- 创建词条
- Markdown 编辑
- 词条之间可以相互链接
- 搜索功能

**学到什么**：内容管理系统基础


## 7. 阅读清单

**难度**：⭐⭐（基础）  
**预计时长**：1-2 小时  
**核心技术点**：列表管理、状态切换

**功能描述**：
- 添加想读的书
- 标记阅读状态（想读/在读/已读）
- 添加读书笔记
- 统计阅读数量

**学到什么**：状态管理、筛选排序


## 8. 写作目标追踪

**难度**：⭐⭐（基础）  
**预计时长**：1-2 小时  
**核心技术点**：目标设定、进度追踪

**功能描述**：
- 设定每日/每周写作目标（字数）
- 记录每天完成的字数
- 显示进度条
- 统计完成率

**学到什么**：进度可视化、数据统计


## 9. 灵感收集箱

**难度**：⭐⭐（基础）  
**预计时长**：1-2 小时  
**核心技术点**：快速记录、标签分类

**功能描述**：
- 快速记录灵感（一句话就行）
- 添加标签分类
- 随机查看一条灵感
- 搜索灵感

**学到什么**：快速输入体验、标签系统


## 10. 文章大纲生成器

**难度**：⭐⭐（基础）  
**预计时长**：1-2 小时  
**核心技术点**：树形结构、拖拽排序

**功能描述**：
- 创建文章大纲
- 多级标题结构
- 拖拽调整顺序
- 导出为 Markdown

**学到什么**：树形数据结构、拖拽交互


## 内容创作者的工具心得

作为内容创作者，最好的工具应该：
- **减少阻力**：打开就能写，不要有复杂的操作
- **专注内容**：功能够用就行，不要干扰创作
- **保护成果**：自动保存，永不丢失

::: tip 建议
如果你是写作者，从「写作计时器」或「每日写作提示」开始。边做边写，一举两得。
:::
